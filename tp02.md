TP2 — Analisis de Set de Datos

1. Kaggle Dataset - Precios de Vuelos
Nombre del producto/servicio: Modelo predictivo de precios de pasajes aéreos basado en dataset de Kaggle.
Qué problema resuelve con ML: Permite predecir el precio de un vuelo a partir de distintas características, ayudando a los usuarios a decidir cuándo comprar y a las empresas a ajustar estrategias de precios.
Qué tipo de ML usa y por qué: Supervisado, específicamente regresión, ya que se busca predecir una variable numérica (precio) a partir de datos previamente etiquetados. El modelo aprende relaciones entre variables de entrada y el valor del pasaje.
Qué datos probablemente necesita para funcionar: Aerolínea, origen, destino, clase del vuelo, días de anticipación de compra, duración del vuelo, cantidad de escalas y horarios. Estas features son útiles porque influyen directamente en el precio final del pasaje.
